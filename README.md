What is that ?
====================
 
Serie of patches and build scripts to build several SDL extension libraries for use with SDL_PSL1GHT. 

Libraries
====================

1. SDL_PSL1GHT + SDL_PSL1GHT_Libs

* sdl - 1.3.0
* sdl_mixer - 1.2.11
* sdl_image - 1.2.10
* sdl_ttf - 2.0.10
* sdl_net - 1.2.7
* sdl_gfx - 2.0.27

2. SDL2_PSL1GHT + SDL2_PSL1GHT_Libs

* sdl2 - 2.31.0
* sdl2_mixer - 2.8.0
* sdl2_image - 2.8.2
* sdl2_ttf - 2.22.0
* sdl2_gfx - 2.22.0
